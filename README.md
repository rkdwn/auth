## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript OIDC-Provider application.

## version info

- yarn v1.22
- Node v18 ~
- node-oidc-provider v7.14 (not latest version)

## Installation

```bash
$ yarn install
```

## Pre-Requisite

you should make your own env files.

please check envs/format.env

## Running the app

```bash
# development(local)
$ yarn start:local

# development
$ yarn start:develop

# production mode
$ yarn start:deploy
```
